# ResuNexx

ResuNexx is an AI-powered resume analysis platform that helps job seekers improve their resumes before applying.

## Features

- AI Resume Analysis
- ATS Compatibility Score
- Resume Quality Score
- Free Resume Preview
- Premium Resume Report
- PDF Report Download

## Tech Stack

- Next.js
- TypeScript
- OpenAI
- Paddle
- Vercel
